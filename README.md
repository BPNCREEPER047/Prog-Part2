# 🛡️ SA Cyber Safety Assistant - ProgPart2

## Cybersecurity Awareness Chatbot for South African Citizens

### 📋 Overview

This application is a **Cybersecurity Awareness Chatbot** designed to educate South African citizens about online safety. It simulates real-life scenarios where users might encounter cyber threats and provides guidance on avoiding common traps.

### 🎯 Features Implemented (Part 2 Requirements)

| Requirement | Implementation Status |
|-------------|----------------------|
| GUI Design & Implementation | ✅ Modern dark theme with gradient brushes |
| ASCII Art Translation | ✅ South African themed ASCII art in welcome message |
| Voice Greeting | ✅ Plays WAV file on launch using SoundPlayer |
| Keyword Recognition | ✅ Detects phishing, password, privacy, malware, 2FA, social media |
| Random Responses | ✅ 8+ variations for each cybersecurity topic |
| Conversation Flow | ✅ Handles "tell me more", "another tip", "explain more" |
| Memory & Recall | ✅ Remembers user name and interests via Dictionary |
| Sentiment Detection | ✅ Detects worried, frustrated, curious sentiments |
| Error Handling | ✅ Graceful handling of missing files, empty input |
